# Mr Robot CTF Writeup

## Overview
This repository contains my walkthrough and security analysis of the Mr Robot Capture The Flag (CTF) machine.  
The project was completed in a controlled lab environment for educational and ethical hacking practice.

---

## Objectives
- Perform enumeration
- Identify vulnerabilities
- Gain initial access
- Escalate privileges
- Capture flags

---

## Skills Practiced
- Network Enumeration
- Web Enumeration
- Vulnerability Assessment
- Password Cracking
- Linux Privilege Escalation
- Security Reporting

---

## Tools Used
- Nmap
- Gobuster
- Nikto
- Metasploit
- Hydra
- Linux Commands

---

## Attack Flow Summary

### 1. Enumeration
Initial scanning and service detection using Nmap.

### 2. Web Enumeration
Discovered hidden directories and robots.txt entries.

### 3. Vulnerability Discovery
Identified vulnerable services and weak credentials.

### 4. Initial Access
Gained shell access through exploitation techniques.

### 5. Privilege Escalation
Escalated privileges to root/admin level.

### 6. Flags Captured
Successfully completed the lab objectives.
